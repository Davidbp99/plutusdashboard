How to Use This Dashboard

  1. Go to https://plutus.it and log in.
  2. Open your browser's developer tools:
    • Windows/Linux: Ctrl + Shift + I or F12
    • Mac: Cmd + Option + I
  3. Navigate to the Console tab.
  4. Paste this code and hit Enter:
(() => {
